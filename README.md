# Automobile Dataset Exploratory Data Analysis (EDA)

A comprehensive data analysis project exploring the automobile dataset using Python and Jupyter Notebook. This project involves data cleaning, visualization, and statistical analysis to uncover insights about vehicle characteristics and pricing.

## Project Overview

This project performs Exploratory Data Analysis (EDA) on an automobile dataset to:
- Clean and preprocess raw data
- Analyze vehicle attributes and their relationships
- Visualize key patterns and trends
- Generate insights for potential business applications

## Dataset Information

### Source File
- **Original File**: `automobile.txt`
- **Format**: Text file with automotive data
- **Contents**: Various attributes of automobiles including pricing, specifications, and features

### Key Attributes
The dataset includes multiple vehicle attributes such as:
- Pricing information
- Technical specifications
- Physical characteristics
- Manufacturer details
